########## Fire | E - Commerce #########

Fire es mi E - commerce que fui creando a partir de lo que iba aprendiendo de los videos
primero preparé todo para un .json el cual interactuaba con el AsynkMock.js que habíamos
usado en primera instancia, después pasé todo el .json a firebase y reemplacé el AsyncMock
por su gestión correspondiente actualizada a la base de datos, para pasar los productos del
.json utilicé un script que encontré en Internet.

El proyecto actualmente en cuanto a layout posee un navbar, una vista para todos los articulos,
otra para categorias especiales, un cartwidget que interactua con el CartContext, y una posibilidad
de checkout y una compra que arroja un Id,
